What Are Examples of Each Type of Chemical Reaction?


Synthesis - C + O₂ ➡ CO₂
Decomposition - CO ➡ C + O
Single Displacement - K + NaCl ➡ KCl + Na
Double Displacement - AgNO₃ + NaCl ➡ AgCl +NaNO₃
